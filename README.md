### 如何运行demo？

1、在项目的根目录执行如下命令
```
$: node index.js
```
2、运行index.html查看控制台的输出


### 基本原理：

index.html内引入了 
```
<script src='./dist/bundle-pro.js'></script>
```

因为我们要手动运行如下命令去生成dist/bundle-pro.js
```
node index.js
```
